Data Science

Article by Shreyas PJ

The world is generating data at a higher rate and so the need for “Data Science” and “Data Analytics” tools increases to analyze and manage this “Big Data”.


Data Science is a field that refers to collective processes, theories, concepts, and tools that enable the review, analysis, and extraction of valuable insights and information from raw data which can be in any form like text, pixel data, or numerical data.

Data Representation

Applications of Data Science

  • Digital Advertisement
  • Internet Research
  • Recommender System
  • Image/speech recognition

Tools and languages used.

  • Python
  • R
  • SAS
  • SQL.

Data Analytics

Data Analytics(DA) is the process of examining data sets to conclude the information they contain, increasingly with the aid of specialized systems and software.

Application areas.

  • Gaming
  • Travel
  • Energy management
  • Healthcare.

Tools and Languages used

  • R
  • Tableau Public
  • Apache Spark for big data analysis.

Big Data

Big Data refers to voluminous amounts of structured data like the information available in form of spreadsheets or tabular columns and unstructured data which is raw data available in form of images, speech, or raw text that organizations can potentially mine and analyze for business gains.

Application areas

  • Communication
  • Retail
  • Financial services
  • Education.

Tools and Languages.

  • Hadoop
  • NoSQL
  • Hive

Most coveted jobs in Data Science.

  • Data Scientist
  • Machine Learning Engineer
  • Machine Learning Scientist
  • Data Architect
  • Data Engineer
  • Business Intelligence(BI) Developer
  • Data Analyst

Even though many of these jobs sound similar, there are a lot of subtle differences between each of them.

Machine Learning Salary in India

The demand for Machine Learning is high but companies require individuals to have the correct skill sets. The demand for these engineers is always high. The list is endless. That’s the primary reason machine learning salary in India is quite high. The ever-increasing demand. Additionally, the better the experience, the higher is the salary. The average machine learning salary in India, according to ‘Payscale’, is approximately Rs. 686,281 per year, inclusive of bonuses and profit-sharing.

Summary

Even though we have seen the distinctions between these fields of work, we realize that they are very closely inter-related to each other and the knowledge in one of these fields will be required in the other 2 fields. So we are working differently with data from different domains and using different tools but they all unite in the fact that they all feed on the same sources of data.
